## Installation

```bash
$ npm install
```

## Connect mongoDB

```bash
paste your mongo database uri into .env file - MONGODBURI
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Use graphql playground

```bash
$ open the browser tab corresponding to your address of the running app(for example - localhost:5000/)
```
